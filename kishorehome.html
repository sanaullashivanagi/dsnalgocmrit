What is Data structures?

Data Sturctures,as the name suggest it is organizing the data in a format so that accessing and manipulating the data becomes esy and efficient.Basic structuring of data comes with setting ordered lists of names or numbers as such. The type of data here is set of characters ie strings, intergers or real numbers (floating value). But whaen it comes to real life problems of handling data, coding for solving real life problems strings intergers will not suffice and requires datatypes which can hold multiple attributes or properties of the datatype instead of having just a name or number. Suppose we need a list of newly released cars with their specifications and the list to be updated. Data Structures provide us the power to solve this requirement in an efficient way by providing the provision to create user defined datatypes. Using this data can be sturctured as per the requirements in a way to increase efficiency of solving the problem reduce size of the code and also reduce excess memory requirements where ever possible.
